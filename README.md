# Cálculo Numérico

### Módulo 1: 
- Método da bisseção;
- Método de Newton-Raphson;
- Método da Secante.
### Módulo 2: 
- Fatoração LU;
- Cholesky;
- Jacobi;
- Gauss-Seidel;
